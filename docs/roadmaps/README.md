# Roadmaps CMDA DEV

Organisation documentaire par roadmap.

- `global` : suivi general R0 a R11.
- `r1` a `r11` : cadrage, etapes, audits et livrables de chaque roadmap.

## Structure

```text
docs/roadmaps/
  global/  suivi global R0 a R11
  r1/      socle API / front
  r2/      authentification JWT
  r3/      routes et roles
  r4/      gestion des erreurs
  r5/      architecture navigation
  r6/      donnees metier et perimetres utilisateurs
  r7/      CRUD structures hierarchiques
```

Les captures et exports visuels restent dans `docs/screenshots`.
